== Quick Start

Add this action to your GitHub workflow:

[source,yaml]
----

on: [push, pull_request]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - name: {{.Name}}
        uses: your-org/{{.Name | lower | replace " " "-"}}@v1
